Run the SEIA:

-1. Make ETOPO1 data in mat format with 'lon', 'lat' and 'topo' or download from http://www.doi.org/10.11922/sciencedb.o00035.00004.

-2. Pre-set pratameters (basically 'main_path', 'yr', 'rslt' and 'area') and run the 'Main_program'.

-3. The function 'SEIA' runs and outputs series of eddies in the file 'Output'.
